▸ 주요 기능
🚨
경보 시스템
비밀번호 오류, 화면 변경, 전원 해제 등 이상 감지 시 즉시 사이렌 경보 발동
📸
침입자 촬영
경보 발동 시 웹캠으로 침입자 자동 촬영 및 타임스탬프 기록
📍
위치 추적
GPS 및 IP 기반 실시간 위치 추적으로 도난 맥북 위치 파악
🔌
전원 감지
충전 케이블 제거 즉시 감지하여 도난 시도 조기 경보
🌐
네트워크 감지
Wi-Fi 변경 감지로 다른 장소로 이동 여부 실시간 모니터링
📋
보안 로그
모든 보안 이벤트를 실시간 기록, 항목별 필터링 설정 지원
🔵
블루투스 감지
등록된 블루투스 기기(이어폰, 마우스 등) 연결이 끊어지면 자리를 비운 것으로 판단하여 즉시 경보 발동
▸ 사용 방법
1
PIN 설정 — 아래에서 6자리 비밀번호를 설정합니다. 잠금 해제 및 설정 변경 시 사용됩니다.
2
시작 버튼 클릭 — 잠금 화면으로 진입하며 보안 감지가 활성화됩니다.
3
맥북을 자리에 두고 자리를 비웁니다 — 이상 징후 감지 시 자동으로 경보가 발동됩니다.
4
돌아왔을 때 — PIN을 입력하면 경보가 해제되고 대시보드에서 보안 로그와 촬영 사진을 확인할 수 있습니다.
