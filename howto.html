// CORE FEATURES

도둑이 건드리는 순간,
모든 것이 발동합니다

MacGuard는 단 하나의 웹 서비스로 작동하는 올인원 맥북 도난 방지 시스템입니다. 설치도, 가입도, 구독도, 무엇보다 비용도 필요 없습니다.

🚨
4종 사이렌 경보
하강·공습·상승·소방차 사이렌이 순차 반복 재생. Web Audio API로 생성된 초고출력 경보음이 최대 볼륨으로 울립니다.
📸
침입자 자동 촬영
비밀번호 오류 즉시 웹캠이 자동 활성화되어 침입자를 촬영합니다. 타임스탬프와 함께 대시보드에 기록됩니다.
📍
IP 기반 위치 추적
IP 기반 위치 추적으로 도난된 맥북의 현재 위치를 실시간으로 파악합니다.
🔌
전원 해제 감지
충전 케이블이 뽑히는 순간 즉시 감지하여 경보를 발동합니다. 도둑이 맥북을 들고 도망치기 전에 사이렌이 울립니다.
🌐
네트워크 변경 감지
연결된 Wi-Fi가 변경되면 즉시 경보. 맥북이 다른 장소로 이동되는 것을 실시간으로 감지합니다.
🛡️
브라우저 닫기 방지
탭 닫기, 창 최소화, 화면 리사이즈까지 모든 회피 시도를 감지하고 차단합니다. 키오스크 모드 지원.

6자리 PIN 설정
나만의 6자리 숫자 비밀번호를 설정합니다. 이 PIN으로 잠금 해제와 모든 설정을 관리합니다.
